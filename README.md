## タイドグラフと天気予報を組み合わせたアプリケーション
釣りで使いやすいように潮位が一目でわかるグラフを配置し潮の流れが速いところは色を変えている。<br>
天気予報を入れることで遠出でも安心！風速と風向きが分かれば遠投時もベイトを狙いやすい！<br>
満潮と干潮の潮位差を出すことである程度の釣り指標にしている。<br>
